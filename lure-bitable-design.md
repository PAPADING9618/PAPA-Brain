# 路亚账号数据追踪 - 飞书多维表格设计

## 表格1：每日数据汇总

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 日期 | 日期 | 数据日期 |
| 发布视频数 | 数字 | 当日发布视频数量 |
| 总播放量 | 数字 | 当日总播放 |
| 总点赞数 | 数字 | 当日总点赞 |
| 总评论数 | 数字 | 当日总评论 |
| 总分享数 | 数字 | 当日总分享 |
| 涨粉数 | 数字 | 当日新增粉丝 |
| 粉丝总数 | 数字 | 累计粉丝数 |
| 备注 | 文本 | 特殊情况说明 |

## 表格2：视频明细数据

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 发布日期 | 日期 | 视频发布日期 |
| 视频标题 | 文本 | 视频标题 |
| 视频链接 | 文本 | 视频URL |
| 播放量 | 数字 | 视频播放量 |
| 点赞数 | 数字 | 视频点赞数 |
| 评论数 | 数字 | 视频评论数 |
| 分享数 | 数字 | 视频分享数 |
| 完播率 | 百分比 | 完播率 |
| 内容类型 | 单选 | 教学/实战/装备/探索/数据/避坑 |
| 表现评级 | 单选 | 爆款/优秀/一般/较差 |
| 优化建议 | 文本 | 基于数据的建议 |

## 表格3：周度复盘

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 周次 | 文本 | 如：2026-W10 |
| 起始日期 | 日期 | 本周开始日期 |
| 发布视频数 | 数字 | 本周发布总数 |
| 总播放量 | 数字 | 本周总播放 |
| 平均播放量 | 数字 | 单视频平均播放 |
| 涨粉数 | 数字 | 本周新增粉丝 |
| 爆款视频数 | 数字 | 播放量>1万的视频数 |
| 最佳内容类型 | 单选 | 本周表现最好的类型 |
| 下周重点 | 文本 | 下周优化方向 |

## 📋 用户操作流程

### 每周操作步骤：

1. **登录抖音创作者中心** (https://creator.douyin.com)
2. **复制以下数据**：
   - 本周发布视频数
   - 各视频的播放量、点赞、评论、分享
   - 完播率数据
   - 粉丝增长数据

3. **发送给我**，我会：
   - 自动写入飞书多维表格
   - 生成数据分析和优化建议
   - 更新长期追踪趋势
