# 🧠 Second Brain | 第二大脑

_papa酱 的记忆管理系统_

## ✨ 功能特性

### 🔍 全局搜索 (Cmd+K)
- 快捷键 `⌘+K` 或 `Ctrl+K` 快速唤起搜索
- 支持模糊搜索，覆盖标题、内容、标签
- 实时显示匹配度和结果预览

### 📋 全量列表
- 按日期分组展示所有记忆
- 支持展开查看详情
- 卡片式极简设计

### 🏷️ 多维筛选
- **类型筛选**：记忆、对话、笔记、任务、目标
- **时间筛选**：全部、今天、本周、本月、今年
- 组合筛选，精准定位

### 🎨 极简设计
- 纯净白色背景
- 精致的排版和间距
- 流畅的交互动画
- 支持暗色模式（TODO）

## 🚀 快速开始

### 安装依赖
```bash
cd /root/.openclaw/workspace/second-brain
npm install
```

### 开发模式
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 📁 项目结构

```
second-brain/
├── app/
│   ├── layout.tsx      # 根布局
│   ├── page.tsx        # 主页面
│   └── globals.css     # 全局样式
├── data/
│   └── memories.ts     # 记忆数据
├── types/
│   └── index.ts        # 类型定义
├── next.config.js      # Next.js 配置
└── package.json        # 依赖配置
```

## 📝 数据类型

| 类型 | 颜色 | 说明 |
|------|------|------|
| 记忆 | 🔵 蓝色 | 长期记忆、重要信息 |
| 对话 | 🟣 紫色 | 对话记录、交流内容 |
| 笔记 | 🟢 绿色 | 学习笔记、整理内容 |
| 任务 | 🟡 黄色 | 待办事项、执行任务 |
| 目标 | 🔴 红色 | 长期目标、规划方向 |

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `⌘/Ctrl + K` | 打开全局搜索 |
| `ESC` | 关闭弹窗 |

## 🔧 技术栈

- **框架**: Next.js 14 + React 18
- **语言**: TypeScript
- **搜索**: Fuse.js（模糊搜索）
- **日期**: date-fns
- **样式**: CSS-in-JS
